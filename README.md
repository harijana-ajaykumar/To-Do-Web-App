# To-Do Web Application

📌 Project Description
The To-Do Web Application is a simple and interactive task management system developed using HTML, CSS, and JavaScript. It allows users to efficiently manage daily tasks by adding, editing, completing, and deleting tasks with date and time support.

---

🚀 Features
- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed
- Add date and time
- Responsive and user-friendly UI

---

 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

 💻 Requirements

## Hardware Requirements

---

▶️ How to Run the Project
1. Download or clone the project folder
2. Open the folder in VS Code
3. Install Live Server extension
4. Open `index.html`
5. Click **Open with Live Server**
